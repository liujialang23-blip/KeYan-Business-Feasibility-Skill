# 可行性检查清单

## 1. 区位与周边配套统计

用于园区、门店、商业空间、文旅、教育培训、酒店住宿、社区服务等项目。

| 类别 | 需统计内容 | 判断意义 | 常见复核点 |
|---|---|---|---|
| 高校/学校 | 学校名称、距离、学生规模、出入口方向 | 青年客群、培训、消费、兼职、创业 | 实际步行路径、校门开放、消费能力 |
| 社区/居住 | 小区、人口、入住率、消费层级 | 日常消费、社区服务、家庭客群 | 真实人口、年龄结构、停车压力 |
| 办公/产业 | 写字楼、园区、企业、员工规模 | 工作日客流、商务消费、招商协同 | 入驻率、上下班时段、企业类型 |
| 商圈/商业 | 商场、街区、餐饮、零售、娱乐 | 竞品、消费氛围、联动机会 | 同质化、租金水平、客流强弱 |
| 文旅/公园 | 景区、公园、湖泊、展馆、艺术区 | 周末客流、打卡、轻消费 | 天气影响、停车、活动管理 |
| 公共服务 | 医院、政务、图书馆、体育馆 | 稳定到访和服务配套 | 与项目业态是否有关 |
| 住宿/酒店 | 酒店、民宿、公寓 | 商务、培训、活动承接 | 淡旺季、价格带、入住率 |
| 停车设施 | 公共停车场、路侧停车、共享停车 | 餐饮、活动、商务到访上限 | 车位数量、收费、可共享时段 |

推荐表格字段：

| 设施名称 | 类型 | 距离/时间 | 可导入客群 | 对项目影响 | 需复核事项 |
|---|---|---:|---|---|---|

## 2. 交通便利性统计

| 交通方式 | 必查内容 | 对项目的影响 |
|---|---|---|
| 地铁 | 最近站点、线路、出入口、步行距离、换乘条件 | 决定跨区域到达便利性 |
| 公交 | 最近站点、线路、首末班、高峰频次、步行距离 | 决定低成本通勤和学生客流 |
| 主干道/快速路 | 主要来向、拥堵、高峰绕行 | 决定机动车到达体验 |
| 高铁/机场 | 距离、时间、换乘方式 | 决定外地商务、活动、文旅客群 |
| 步行 | 人行道连续性、过街、夜间照明、雨天路径 | 决定真实可达性 |
| 非机动车 | 自行车、电动车停放、充电、安全 | 决定高校、社区和员工高频到达 |
| 网约车/出租车 | 落客点、回车、夜间照明、拥堵 | 决定商务接待和夜间消费 |
| 停车 | 现状车位、周转率、共享停车、消防车道 | 决定餐饮、活动、培训和商业上限 |
| 后勤 | 装卸、垃圾清运、配送车、消防车 | 决定运营稳定性和安全 |

不要只写“交通便利”。必须写出便利在哪里、不便在哪里、哪些客群因此能来、哪些业态会被限制。

## 3. 场地与硬性条件

| 专项 | 必查内容 | 风险 |
|---|---|---|
| 权属用途 | 不动产权、土地用途、租赁授权、房屋用途 | 无法办证、无法装修、无法申报 |
| 结构荷载 | 建成年代、结构形式、楼板荷载、裂缝沉降 | 重设备、密集人流、机房、仓储风险 |
| 消防 | 安全出口、疏散宽度、防火分区、喷淋烟感、消防车道 | 培训、餐饮、住宿、活动无法开展 |
| 电力 | 变压器、楼栋容量、扩容路径、备用电 | 直播、厨房、机房、设备受限 |
| 给排水 | 卫生间、厨房、洗衣、排污、雨污分流 | 餐饮、住宿、培训承载不足 |
| 排烟排污 | 油烟、隔油池、垃圾、噪声 | 重餐饮和夜间商业受限 |
| 暖通网络 | 空调、新风、散热、光纤、专线、弱电井 | 数字企业、直播、培训体验受限 |
| 安防运营 | 门禁、监控、夜巡、湖边/街边安全 | 夜间运营、住宿、活动风险 |

## 4. 商业模式检查

- 主要收入是否超过一种；
- 核心收入是否依赖尚未验证的客流或政策补贴；
- 成本是否包含人力、营销、运维、合规、维修、空置期；
- 是否有低成本验证路径；
- 用户付费动作是否清晰；
- 谁负责日常运营；
- 指标是否能在 30/90/180 天内观察。

## 5. 政策与合规检查

按项目类型选择：

| 项目类型 | 合规重点 |
|---|---|
| 餐饮/食堂 | 食品经营许可、排烟、排污、消防、垃圾清运 |
| 住宿/公寓/民宿 | 房屋性质、消防、公安、卫生、经营许可 |
| 培训/教育 | 办学许可、消防、人流、师资、收费规范 |
| 活动/展演 | 人员峰值、临时搭建、消防、公安、文旅审批 |
| AIGC/内容 | 生成式AI、深度合成、版权、内容标识、数据安全 |
| 医疗/健康 | 医疗资质、广告合规、人员资质、隐私 |
| 金融/交易 | 金融许可、反洗钱、投资者适当性、广告 |
| 儿童/老人 | 安全、人员资质、保险、应急预案 |

## 6. 风险清单模板

| 风险 | 等级 | 影响 | 触发条件 | 控制措施 |
|---|---|---|---|---|
| 消防不达标 | 高 | 无法开放培训、餐饮、活动 | 消防评估不通过 | 先评估再招商，限制人数 |
| 停车不足 | 中高 | 餐饮和活动客流受限 | 高峰占道或投诉 | 共享停车、预约、轻业态 |
| 招商不足 | 中高 | 出租率和现金流不足 | 意向面积低于目标 | 扩大替代客群，先做样板 |
| 政策申报不确定 | 中 | 补贴不能兑现 | 指标或主体不匹配 | 不把补贴作为基础收益 |

## 7. 资料补充清单

| 类别 | 资料 |
|---|---|
| 地址与权属 | 精确地址、坐标、红线、权属、租赁授权、用途 |
| 建筑与结构 | 总平、建筑图、结构图、竣工图、荷载、检测 |
| 消防与安全 | 消防验收/备案、消防车道、安全出口、应急预案 |
| 机电与网络 | 电力容量、给排水、排烟、空调、新风、光纤 |
| 交通停车 | 车位、出入口、公交地铁、共享停车、落客点 |
| 市场客群 | 客流数据、竞品、租金、价格、访谈、消费能力 |
| 运营财务 | 租金、改造预算、人员、渠道、成本、收入假设 |
| 证照政策 | 许可证、备案、政策申报通知、主管部门口径 |

## 8. 入驻或商户准入资料

| 类别 | 需提交内容 |
|---|---|
| 主体 | 营业执照、经营范围、实际经营主体 |
| 业态 | 是否餐饮、住宿、培训、直播、AIGC、活动、零售 |
| 空间 | 面积、装修平面、设备布置、消防疏散 |
| 设备 | 设备清单、功率、用水、用气、排烟、空调 |
| 人流 | 员工、峰值客流、营业时间、活动频次 |
| 交通 | 停车、配送、网约车、非机动车需求 |
| 合规 | 许可证、备案、审批、承诺文件 |
